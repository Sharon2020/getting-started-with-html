THE BASICS OF WEB AND HTML

The main type of document on the world wide web is HTML (hyper text markup language). The type of files found on the web include plain text, HTML, videos, and music. Your browser sends requests to the internet which uses HTTP Protocol to communicate with servers. HTTP is the main protocol of the web. Servers are computers that host the files that make up the web. The Internet is the worlds largest computer. Your browser is the program that runs on your computer to display files found on the web.

HTML is made up of text content (what you see), markup (what it looks like), references to other documents (images and videos), and links to other pages.
Tags have an opening and closing tag with the content in the middle and the whole thing is called an element. Tags can include attributes as well. A void tag has no contents and does not need a closing tag. Span and Div elements only contain their contents and attributes can change the style.
Inline elements are just text and block elements actually make a box that can have a height and width associated with it. a, span, br, img, and strong are inline elements. div, p, and form are block elements.

TAGS AND ATTRIBUTES:
B TAG – BOLD <b> contents </b>
EM TAG – Emphasis or italics <em>contents</em>
Using attributes: <tag ATTR=”value”>contents</tag>
ANCHOR TAG - To make links <a> contents </a>
Example: <a href=www.reddit.com>derp</a> Will display derp as a hyperlink.
IMG TAG – Images <img src=”url” alt=”text”> Alt is required to show what to display if it doesn’t work.
BR TAG - Page break. (Void tag) <br>
PARAGRPAH TAG - Marks paragraph. <p>content</p>
